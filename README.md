#ezo
#easily create Ethereum oracles

# ### this repo is incomplete ###

Initialial Installation
`ezo create` 

Compile Contract
`ezo compile <contract file>`

Deploy Contract
`ezo deploy <contract hash> -s <stage>`

Generate Handler Scaffold
`ezo gen <contract hash>`

View Contracts
`ezo view contracts`

View Deployments
`ezo view deployments`

Start ezo
`ezo start`
